How It Works
Four steps. One command each.
COLT learns your app by exploring it — then executes anything you describe.
01

Discover your app

Autonomously crawls your web application. Clicks buttons, opens modals, switches tabs, scrolls lists. Maps every page, form, and interactive element with exact CSS selectors. Handles Shadow DOM and cross-origin iframes.

BFS crawlShadow DOM piercingIframe traversalScroll detectionPagination guard
02

Build a knowledge base

Each state is summarized by an LLM and stored as a vector embedding. Creates a persistent, searchable index of your entire app. Discover once — then run unlimited tasks without re-crawling.

LLM summariesVector embeddingsChromaDBState graphAPI schema capture
03

Execute any task

Describe what you want in plain English. A ReAct agent observes live DOM state, reasons about the next action, and executes it. Self-heals on failures with fallback chains. Detects and breaks out of infinite loops.

Observe / Reason / ActSelf-healingLoop detectionToken budget
04

Export reusable code

Every action is traced with full context and exported as standard Playwright Python. Run the output without COLT, modify it by hand, or commit it to your repo. No vendor lock-in.

Playwright scriptPytest testTyped Python functionsJSON trace
Why COLT
Built different from other browser agents
Most AI agents start from scratch every run and produce nothing reusable. COLT remembers your app and outputs real code.

Persistent knowledge

One crawl builds a knowledge base. Task #100 executes as fast as task #1 — no re-exploration, no wasted LLM calls.

Code output, not just execution

Every run produces a standalone Playwright script. Run it without COLT, modify it, commit it. The automation outlives the agent.

Handles complex UIs

Shadow DOM components, cross-origin iframes, infinite scroll, modals, tabs, pagination — the edge cases that break other tools.

Documentation
Get started in 60 seconds
Install, initialize, discover, execute.

Installation

Python 3.10+ required. Node.js for Playwright browsers.

# Install pip install colt-automation playwright install chromium # Set your LLM key export GROQ_API_KEY=your-key-here

Initialize a Project

Generates config, output directory, installs browsers.

# Interactive setup colt init my-app --url http://localhost:3000 # With specific LLM colt init my-app --url http://localhost:3000 --llm openai

Discover Your App

Crawls the application and builds the knowledge base.

# Headless colt discover # Watch in browser colt discover --no-headless # Custom config + depth colt discover -c myapp.yaml -d 3

Execute and Record

Run tasks and export Playwright scripts.

# Execute with ReAct agent colt execute "Create admin user" --react # Record to script colt execute "Create admin user" \ --react --record replay.py # Generate typed functions colt generate automation.py

CLI Reference

colt init <name> --url <url> --llm <provider>Initialize project with config + browser setup
colt discover [-c config.yaml] [--no-headless]Crawl app and build knowledge base
colt execute "task" --react [--record file.py]Execute task with ReAct agent, optionally record
colt generate output.py [--dry-run]Generate typed functions from discovered forms
colt record trace.json output.py --format <fmt>Convert execution log to script / pytest / json / all
colt search "query" [-n 5]Search knowledge base for matching pages
